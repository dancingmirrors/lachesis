lachesis
========
This is a standalone fork of ffplay with a modified version of the build system from mpv's bliss branch.

Dependencies include FFmpeg 7.1 or greater, SDL3, SDL_ttf, libarchive, Vulkan development headers and  
libraries, libplacebo 7.349.0 or greater, and (probably) GCC or Clang. The only optional dependency at  
the moment is yt-dlp. This was more or less based around what Debian Trixie provides.
